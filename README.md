# Ford-Car-Price-Prediction-App
# 🚗 Ford Car Price Prediction Web App

## 📌 Project Overview
The Ford Car Price Prediction Web App is a Machine Learning project that predicts the estimated selling price of a Ford car based on user inputs. The application is built using Streamlit and a Linear Regression model trained on the Ford Used Car Dataset.



## 🎯 Aim
To develop a web application that predicts the selling price of Ford cars using Machine Learning.

---

## 📖 Features
- Predicts Ford car selling price instantly
- User-friendly Streamlit interface
- Supports different Ford car models
- Numerical and categorical input fields
- One-Hot Encoding for categorical data
- StandardScaler for feature scaling
- Linear Regression prediction model
- Clean and responsive interface

---

## 🚀 How to Run

### Step 1: Clone the Repository

```bash
git clone https://github.com/vaishnavi-chandak4/Ford-Car-Price-Prediction-App.git
```

### Step 2: Open the Project Folder

```bash
cd Ford-Car-Price-Prediction-App
```

### Step 3: Install Required Libraries

```bash
pip install -r requirements.txt
```

### Step 4: Run the Streamlit App

```bash
streamlit run app.py
```
